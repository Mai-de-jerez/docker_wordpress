# 🚀 Proyecto Morfeo - WordPress Full Stack Profesional

Este proyecto es un despliegue optimizado de WordPress con Docker. Está configurado para que cualquier desarrollador pueda clonarlo y tener el entorno funcionando idéntico al original en cuestión de segundos.

## 🛠️ Requisitos
* **Docker** instalado.
* **Docker Compose** instalado.

## 📥 Cómo ponerlo en marcha (Instrucciones para mi compi)

1. **Clona el repositorio en tu máquina Linux:**
   ```bash
   git clone [https://github.com/Mai-de-jerez/docker_wordpress.git](https://github.com/Mai-de-jerez/docker_wordpress.git)
   cd docker_wordpress

### 🚀 Levantando el entorno

Para poner en marcha la web, ejecuta el siguiente comando en tu terminal:

```bash
sudo docker compose up -d
